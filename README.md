# Commands

:heart: One-By-One :heart:

```bash

git clone https://github.com/Zycho-Dev-66/Zyntax-Banner

cd Zyntax-Banner

chmod +x *

bash Banner.sh

```

:heart: ONE-LINE COMMAND

```bash
git clone https://github.com/Zycho-Dev-66/Zyntax-Banner && cd Zyntax-Banner && chmod +x && bash Banner.sh
```
